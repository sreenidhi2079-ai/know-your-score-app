Project Overview

Know Your Score is an advanced AI-driven diagnostic tool designed to predict and analyze smartphone addiction levels. By correlating daily screen time, interaction frequency, and offline hours, the system generates targeted insights based on user behavior.

Technical Stack

  • Algorithm: Decision Tree Classifier
  • Backend: Flask (Python)
  • Frontend: HTML5, CSS3 Custom Properties
  • Classification Output: Low / Medium / High
